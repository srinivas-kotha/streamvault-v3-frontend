# StreamVault v3 Frontend

TV-first IPTV frontend for Fire TV Stick 4K Max. React 19 + Vite 5 + Tailwind 4 + norigin-spatial-navigation 2.1.0.

Private repo; open-source ready later. See `docs/DECISIONS.md` for locked architectural decisions and `docs/KNOWN-RISK.md` for tracked risks.

## Scripts

- `npm run dev` — local dev server
- `npm test` — unit tests (vitest)
- `npm run typecheck` — TypeScript compile check
- `npm run lint` — ESLint
- `npm run build` — production build
- `npm run check-budget` — bundle size guard (post-build)
